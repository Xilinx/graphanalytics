Xilinx Maximal Independent Set for TigerGraph 3.x
===============================================

Xilinx Maximal Independent Set for TigerGraph 3.x is a plug-in for `TigerGraph 3.x 
<https://www.tigergraph.com>`_ that adds user-defined functions that you can use in 
your own GSQL queries to run `maximal independent set <https://en.wikipedia.org/wiki/Maximal_independent_set>`_ 
of vertices in your graph data using a Xilinx Alveo accelerator cards.
