Xilinx Fuzzy Match for TigerGraph 3.x
===============================================

Xilinx Fuzzy Match for TigerGraph 3.x is a plug-in for `TigerGraph 3.x 
<https://www.tigergraph.com>`_ that adds user-defined functions that you can use in 
your own GSQL queries to determine if two strings match based on the similarity score
computed from Levenshtein edit distance using :ref:`fuzzymatch-label`.
