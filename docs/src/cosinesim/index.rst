Xilinx Cosine Similarity Alveo Product
======================================

The Cosine Similarity Alveo Product allows you to use a Xilinx Alveo accelerator card to find the best matches
of a given vector of integers within a large set of such vectors using the
`cosine similarity <https://en.wikipedia.org/wiki/Cosine_similarity>`_ formula on the vectors.
