Xilinx Louvain Modularity Alveo Product
=======================================

The Louvain Modularity Alveo Product allows you to run 
`Louvain method <https://en.wikipedia.org/wiki/Louvain_method>`_ on a Xilinx 
Alveo accelerator card to extract communities from large networks.
