Xilinx Maximal Independent Set Alveo Product
============================================

The Maximal Independent Set Alveo Product uses a Xilinx Alveo accelerator card to find the `maximal independent 
set <https://en.wikipedia.org/wiki/Maximal_independent_set>`_ of vertices in a graph.
